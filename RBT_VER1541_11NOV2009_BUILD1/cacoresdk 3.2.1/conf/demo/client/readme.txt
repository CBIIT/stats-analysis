To build and run the TestClient.java file.
1. Set ant to your path variable
2. Unzip client.zip in a new directory such as c:/cagwas-client
3. Modify the build.properties file so that 
	a) web_server_name = points to the the remote or local server name or URL
	b) web_server_port = points to the server port
4. Save build.properties
5. Run ant from the unzipped directory
